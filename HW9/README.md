# README

## How to use
Type in console:

```ruby
git clone https://github.com/Haidamac/RubyHW.git
```

```ruby
cd RubyHW/HW9
````

```ruby
gem install bundler
```
```ruby
bundle install
```

Attach ```config/database.yml``` acording `database copy.yml` with your username and password

Type in console:

```ruby
rails db:create
```

```ruby
rails db:migrate
```

```ruby
rails db:seed
```

```ruby
rails s
```

Open Swagger in browser:
```ruby
http://127.0.0.1:3000/api-docs/index.html
```
