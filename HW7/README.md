# README

## How to use
Type in console:

```ruby
git clone https://github.com/alinajuly/RubyHW.git
```

```ruby
cd RubyHW/HW7
````

```ruby
gem install bundler
```
```ruby
bundle install
```

Attach ```config/database.yml``` acording `database copy.yml` with your username and password

Type in console:

```ruby
rails db:create
```

```ruby
rails db:migrate
```

```ruby
rails db:seed
```

```ruby
rails s
```
